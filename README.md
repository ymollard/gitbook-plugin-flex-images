# Flexbox Thumbnails for GitBook

Display a block image images as a flexible row of thumbnails.

**Note** This does not yet work with ebook (pdf, ePub) output.

```markdown
{% fleximages %}
  ![Alt Text](http://placehold.it/640x480)
  ![Alt Text](http://placehold.it/640x480)
  ![Alt Text](http://placehold.it/640x480)
  ![Alt Text](http://placehold.it/640x480)
{% endfleximages %}
```
